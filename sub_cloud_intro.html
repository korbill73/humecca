서비스 개요

Server는 안전하고 크기 조정이 가능한 컴퓨팅 인프라를 가상화하여 제공하는 서비스입니다.
웹 인터페이스나 API를 통해 필요한 서버 인프라를 간편하게 구성할 수 있습니다.
해당 서비스로 구성된 서버들의 리소스는 kt cloud에서 안전하게 관리하며, 고객 단독 사용이 보장됩니다.

서비스 특징

확장성

· Open API/SDK 제공
· 외부 서비스 빠른 개발
· Incubation Center 지원

안정성

신속성

고성능

· CPU/IOPS 벤치마크 상위
· 40Gbps 백본 품질 보장

보안

· 이중화된 백본망 재난 대비

서비스 구성 (서버 세대)

1세대 서버 구성

기존세대 Haswell CPU까지 적용되는 1세대 서버입니다. 고성능 I/O 서버 필요시 SSD Server를 신청 가능합니다.

2세대 서버 구성

Skylake, Broadwell 기반의 고성능 CPU 적용되는 2세대 서버 입니다. 기본적으로 SSD 루트 디스크가 제공되며, 고성능 또는 비용 최적화를 위해 데이터 디스크를 자유롭게 선택 가능합니다.

3세대 서버 구성 (New)

Cascade 기반의 고성능 CPU 적용되는 3세대 서버 입니다. Openstack플랫폼을 기반으로 고성능 VM을 제공합니다. VM당 최대 64vCore, 256GB메모리를 사용할 수 있습니다.

주요 기능

VM 구성 및 관리

  • 가상화된 VM 생성 (CPU/Disk 자유 조정)
  • 최적화된 VM 템플릿 제공

Network 제어

  • 방화벽 설정 및 포트포워딩
  • WAF 연동 보안 강화
  • 내부망(CIP) 및 LB 구성

백업과 이중화

  • 스냅샷/이미지 백업 및 복원
  • 디스크, 백본망, 서버(HA) 이중화

Monitoring

  • 로그 히스토리/트래픽 통계
  • 리소스 사용량 확인 및 알람(Watch)
고객 성공 사례

금융 증권사 솔루션 사례

  • Needs: 실시간 시세 제공 안정성
    트래픽 급변 대응
  • Solution: API 스케줄링 자동화
    Load Balancer 트래픽 분산

기대효과: 예상치 못한 트래픽/장애에 빠른 대응 가능

  • Needs: 대량 VM 및 부하분산

  • Solution: 40대 서버 구성, CIP 연동
    WAF 방화벽 설정

주요 부가 서비스
  • API 서비스: Amazon EC2와 유사한 API 제공 (VM 관리, 볼륨 제어 등)
  • 스냅샷/이미지: 디스크 상태 저장 및 시점 복원을 위한 백업 서비스
  • CIP (Cloud Internal Path): 내부망 구성을 위한 별도 Network 환경 제공
  • Scale up/down: 운용 중인 서버 사양 유연한 변경 (vCore, Memory 조정)
